Scheme Markup Experiments with fizzbuzz.scm

Here are some experiments in marking up the simple Scheme program fizzbuzz.scm
to explore possible ways to increase the readability of programs represented as nested lists.

fizzbuzz.scm
Scheme source code with standard indentation.
fizzbuzz1.html
Scheme source code with simple HTML markup by the highlight program.
fizzbuzz2.html
An experiment in markup with parentheses
  1. changing color
  2. decreasing in size
with nesting level. Also, parentheses around special forms are in bold.
fizzbuzz8.html
An experiment in markup replacing parentheses around function definitions
and parentheses spanning multiple lines with inset shadowed boxes.
Thanks to Alex Turner for the great shadow boxes!
